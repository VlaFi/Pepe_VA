Pepe Voice Assistant

Экспериментальный голосовой ассистент на базе `llama-cpp-python` с поддержкой GPU.

Текущее состояние
Проект на паузе.  
Сейчас реализовано:
- Базовая структура модулей (`core`, `hardware`, `llm`);
- Инициализация `llama-cpp-python` с проверкой GPU;
- Автоматический выбор оптимальных параметров для запуска модели;
- Поддержка виртуального окружения Python (`.venv`).

План на будущее
- Добавить голосовой ввод/вывод;
- Расширить модуль LLM (контекст, память, диалог);
- Сделать веб-интерфейс для общения;
- Оптимизация под RTX 4060 GPU.

Как запустить
1. Создать виртуальное окружение:
   ```bash
   python -m venv .venv
   .venv\Scripts\activate
2. Установить зависимости:

pip install -r requirements.txt

3. Запустить в терминале:

python main.py
